еҰӮдҪ•еҸ‘еёғдҪ иҮӘе·ұзҡ„и„ҡжң¬

2025-07-01 · 1 min read · python

  дҪ жҳҜдёҚжҳҜд№ҹжғіи®©иҮӘе·ұзҡ„зЁӢеәҸеғҸobspyдёҖж ·пјҢеҸ«е…Ёдё–з•Ңзҡ„дәәpip installе°ұеҸҜд»Ҙз”Ёпјҹ

  иҝҷйҮҢд»ҘдёҖдёӘз®ҖеҚ•зҡ„дҫӢеӯҗжқҘиҜҙжҳҺеҰӮдҪ•еҲӣе»әиҮӘе·ұзҡ„зЁӢеәҸгҖҒжү“еҢ…е№¶еҸ‘еёғеҲ°PyPIгҖӮ

еҲӣе»әиҮӘе·ұзҡ„йЎ№зӣ®

йҰ–е…ҲеҲӣе»әдёҖдёӘзӣ®еҪ•mypkg_projectпјҢз»“жһ„еҰӮдёӢпјҡ

mypkg_project/
в”ңв”Җв”Җ mypkg/
в”Ӯ   в”ңв”Җв”Җ __init__.py
в”Ӯ   в””в”Җв”Җ utils.py
в”ңв”Җв”Җ README.md
в”ңв”Җв”Җ setup.py
в”ңв”Җв”Җ pyproject.toml

зј–еҶҷеҢ…зҡ„еҶ…е®№

еңЁmypkg/utils.pyеҶ…еҶҷе…Ҙпјҡ

def add(a, b):
    return a + b

еңЁmypkg/init.pyеҶ…еҶҷе…Ҙпјҡ

from .utils import add

ж·»еҠ е…ғж•°жҚ®ж–Үд»¶README.md

еңЁREADME.mdдёӯеҶҷе…Ҙпјҡ

# mypkg
A simple Python package with an add function.

еңЁsetup.pyдёӯеҶҷе…Ҙпјҡ

from setuptools import setup, find_packages
setup(
    name='mypkg',
    version='0.1',
    author='Your Name',
    description='A simple example package',
    long_description=open('README.md').read(),
    long_description_content_type='text/markdown',
    packages=find_packages(),
    classifiers=[
        'Programming Language :: Python :: 3',
    ],
    python_requires='>=3.6',
)

еңЁpyproject.tomlдёӯеҶҷе…Ҙпјҡ

[build-system]
requires = ["setuptools", "wheel"]
build-backend = "setuptools.build_meta"

йЎ№зӣ®жү“еҢ…

еңЁж №зӣ®еҪ•mypkg_project/дёӯиҝҗиЎҢпјҡ

python3 -m pip install --upgrade build
python3 -m build

дјҡз”ҹжҲҗпјҡ

dist/
в”ңв”Җв”Җ mypkg-0.1-py3-none-any.whl
в””в”Җв”Җ mypkg-0.1.tar.gz

жң¬ең°е®үиЈ…

з”Ёpipе°ұеҸҜд»Ҙжң¬ең°е®үиЈ…пјҡ

pip install dist/mypkg-0.1-py3-none-any.whl

еҲ©з”Ёpythonе°ұеҸҜд»Ҙи°ғз”Ёпјҡ

from mypkg import add
add(2, 3)

дёҠдј еҲ°PyPIпјҲе…¬ејҖеҸ‘еёғпјү

йҰ–е…Ҳе®үиЈ…twineпјҡ

pip install twine

еҸ‘еёғпјҡ

twine upload dist/*

иҝҷйҮҢдјҡжҸҗзӨәдҪ иҫ“е…ҘPyPIзҡ„з”ЁжҲ·еҗҚе’ҢеҜҶз ҒгҖӮиҝҷдёҖжӯҘйңҖиҰҒйҰ–е…ҲеҲ°PyPIжіЁеҶҢиҙҰеҸ·пјҢеҸ‘еёғзҡ„ж—¶еҖҷйңҖиҰҒз”Ёtoken

twine upload --repository pypi dist/* -u __token__ -p pypi-<your_token>

дёҠдј жҲҗеҠҹд№ӢеҗҺеҸҜд»ҘеңЁhttps://pypi.org/project/mypkg/и®ҝй—®гҖӮ з„¶еҗҺд»»дҪ•дәәйғҪеҸҜд»Ҙе®үиЈ…дҪ зҡ„и„ҡжң¬дәҶпјҡ

pip install mypkg
