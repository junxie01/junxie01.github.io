bash里如何循环读取文件

2021-03-09 · 1 min read · Linux

  Bash操作简单，可以处理简单的计算等。有些时候会遇到文件的读取问题，那如何对文件进行循环逐行读取呢？

  这样就可以了:

while read -r line
do
   array=(${line///})  
   echo ${array[1]}
done < the_file
